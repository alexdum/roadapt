# RO-Adapt (Romania)

This project contains the Shiny application that powers the interactive map for the [RO-Adapt Geoportal](https://www.roadapt.ro/geoportal-harta-interactiva.php), a national climate platform for Romania. 

The RO-Adapt platform provides interactive access to climate parameters and indicators attributed to various economic sectors in Romania, including:
- Agriculture and Rural Development
- Energy
- Water Resources
- Forestry
- Biodiversity and Ecosystem Services
- Cultural Heritage
- Urban Systems
- Transport
- Tourism and Recreational Activities
- Public Health
- Industry
- Insurance

The geoportal serves as a central hub for climate data, advanced mapping, data catalogs, and resources for climate adaptation strategies.
